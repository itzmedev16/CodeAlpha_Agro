# CodeAlpha_Agro
This agri-based e-commerce web application connects users with agricultural products through a simple online platform. It includes a secure login system, a user-friendly index page for browsing and adding products to the cart, and an admin dashboard to manage products, stock, and inventory efficiently.

🌾 CodeAlpha Agro - Agri E-Commerce Website

Description
  CodeAlpha Agro is an agriculture-based e-commerce web application designed to connect users with agricultural products through a simple and user-friendly platform. It allows users to browse, select, and purchase products efficiently while enabling administrators to manage inventory.
  
Features
*  User Authentication (Login/Register)
*  Add to Cart Functionality
*  Product Listing & Categories
*  Admin Dashboard
*  Stock Management (In-stock / Out-of-stock)
*  Product Add / Delete / Update

 Technologies Used

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express.js
* Database: MongoDB

Project Structure

* backend` → Server, routes, models
* css` → Styling files
* js` → Client-side scripts
* pages` → HTML pages
* uploads` → Product images

How to Run the Project
Prerequisites
Make sure you have installed:
* Node.js
* MongoDB (local or MongoDB Atlas)

Steps to Run
1.Clone the repository
git clone https://github.com/itzmedev16/CodeAlpha_Agro.git
cd CodeAlpha_Agro
2.Install backend dependencies
cd backend
npm install
3.Start the server
node server.js
4.Open the application
* Open `index.html` in your browser
  OR
* Use Live Server (VS Code recommended)
Server Runs On

http://localhost:5000
Notes
* Ensure MongoDB is running before starting the server
* Update database connection in `server.js` if needed
* Do not upload `node_modules` folder
